## 简介
cloud-media-sync（CMS）--- 云端媒体库同步工具

> 监控`115`文件夹，生成`emby`可以识别的`strm`文件，支持增量，支持`emby302`。

> 本项目基于 [`python-115`](https://github.com/ChenyangGao/web-mount-packs)，懂代码的可以看看，非常强大。

> 致谢：`尽贫·禁评` `DDSRem`

> 致谢：`Black_Plum` `HuLuXi` 提供相关容器安装环境教程
